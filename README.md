# Cognitive UI

## Setup

```bash
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
yarn dev
```

## Production

Build the application for production:

```bash
yarn build
```

## Lint

Manual run lint fix:

```bash
yarn lint:fix
```

## ESLint config inspector

Start the development ESLint config inspector on `http://localhost:7777`:

```bash
yarn lint:inspector
```
